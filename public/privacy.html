🔒 Privacy Policy

We built this tool to be useful, not to collect your data. Here's exactly what we do and don't do.

Short version: We don't sell your data, run ads, or track you. Car selections and product lists live on your device only. The only external service used is Anthropic's API for dimension extraction — and only as a fallback on sites that don't publish structured data.

🏠 Overview

Will it Fit? helps you check whether furniture or large items will fit in your car's boot before driving to a store. This policy covers both the web app at will-it-fit-livid.vercel.app and the Chrome browser extension.

📋 What we collect

1. Product page text (temporary, never stored by us)

When you visit a product page and the extension or web app cannot extract dimensions automatically from the page's structured data, a portion of the page's visible text (up to 10,000 characters) is sent to the Anthropic API for dimension extraction.

  • This text is used solely to identify numbers — packaging dimensions.
  • It is not stored by us on any server.
  • This only happens on sites without structured product data. IKEA, Home Depot, Wayfair, and Amazon are handled locally with zero API calls.
  • Anthropic's handling of this data is governed by their Privacy Policy.

2. Your Anthropic API key (extension only, stored locally)

If you provide an Anthropic API key in the extension settings, it is stored in Chrome's encrypted local storage (chrome.storage.local) on your device only. It is transmitted only to api.anthropic.com when making dimension requests — never to our servers or any other party.

3. Car selection (stored locally)

Your selected car make, model, and year are saved locally (in the browser's localStorage for the web app, or chrome.storage.local for the extension) so the tool remembers your car between sessions. This never leaves your device.

4. Product list (stored locally, session only)

Products you add to your fit-check list are held in local session storage. This data is not transmitted anywhere and is cleared when you close the tab or uninstall the extension.

🚫 What we don't collect

None of the following are collected, stored, or shared — ever.
  • Your name, email address, or any personally identifiable information
  • Browsing history or a list of sites you visit
  • Cookies or tracking pixels of any kind
  • Analytics on how you use the tool (no Mixpanel, GA4, Amplitude, etc.)
  • Device fingerprints, IP addresses, or location data
  • Any data shared with advertisers (we have no advertisers)

🔗 Third-party services

Anthropic API

When package dimensions cannot be extracted from a page's structure directly, visible product page text is sent to Anthropic's API for processing. This is the only external service we use.

  • Data sent: a text excerpt (≤10,000 characters) from the current product page
  • Purpose: extracting numerical package dimensions
  • Anthropic's privacy policy: anthropic.com/privacy

Affiliate links (when enabled)

A future version of the extension may include affiliate links to retailers. When enabled, clicking a product link may route through an affiliate tracking URL. This will be clearly disclosed in the interface. The respective retailer's privacy policy governs any data collected after you click through.

Vercel (web app hosting)

The web app is hosted on Vercel. Vercel may log standard server access data (IP, timestamp, referrer) as part of their infrastructure. See Vercel's Privacy Policy.

🔑 Extension permissions explained

The Chrome extension requests only the permissions necessary to function:

PermissionWhy it's needed
activeTab Reads the text of the product page you're currently viewing to extract packaging dimensions
tabs Detects when you navigate to a new page so the panel can auto-refresh with the new product's dimensions
storage Saves your car selection and API key locally on your device between sessions
sidePanel Displays the tool in Chrome's native side panel UI
scripting Injects a lightweight script that reads packaging dimension text from product pages
<all_urls> The extension must work on any retailer's website; the list of furniture retailers cannot be enumerated in advance

🗑️ Data retention and deletion

All data stored by this tool is on your device. You can delete it at any time:

Web app

  • Open your browser's developer tools → Application → Local Storage → clear entries for this site
  • Or simply clear your browser's site data

Chrome extension

  • Open the extension panel → click the settings icon → Options → "Clear Saved Key & Car"
  • Or uninstall the extension — all associated storage is removed automatically

👶 Children's privacy

This tool is not directed at children under 13. We do not knowingly collect any personal information from children. If you believe a child has provided personal information through this service, please contact us and we will delete it promptly.

📝 Changes to this policy

If we make material changes to this privacy policy, we will update the "Last updated" date at the top of the page. Continued use of the tool after changes are posted constitutes acceptance. We will not retroactively apply changes to data already collected under a prior policy.
